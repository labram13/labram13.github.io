# GreatFrontEnd Projects Profile Card Challenge

## Tech stack and approach
Tailwind CSS and flex box approach to style the card. Provided Figma files used to apply exact styling such as font-sizes and spacing.


## Useful resources and lessons learnt
When adding background images to the top portion of the card, it's important to match rounded corner styling to the card as well as classes done to the parent does not affect image backgrounds. 



## Steps to run program

1. git clone repository
2. Open folder in VSCode
3. Go to Extensions tab
4. Search and install Live Server
5. Right click index.html file and select Open Live Server to view in web browser.
